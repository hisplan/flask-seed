# flask-seed

## Prerequisits

```bash
$ sudo pip install flask
```

## Run

```bash
$ export FLASK_APP=my_web.py
$ export FLASK_DEBUG=1
$ flask run --port 5000
```

## Unit Test

```bash
$ python test_app.py
```
